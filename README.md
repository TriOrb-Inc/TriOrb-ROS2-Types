# TriOrb-ROS2-Types
TriOrb AMRのROS2パッケージのためのInterface

# Environment setup
```bash
sudo apt install libssl-dev
```

# Clone and install
```bash
cd ~/
git clone https://github.com/TriOrb-Inc/TriOrb-ROS2-Types.git
cd ~/TriOrb-ROS2-Types
colcon build &&\
source install/setup.bash
```
